# 🛠️ 软件工程

跨语言、跨端的通用工程实践，关注代码之外的质量保障。

## Web 安全

XSS、CSRF、SQL 注入、DDoS——了解攻击手段才能有效防御。HTTPS 与 TLS 为传输安全提供基础保障，CSP、SameSite 等现代浏览器机制进一步加固防线。

## 设计模式

SOLID 原则与常用设计模式在真实业务中的应用，让代码更可维护、可扩展。

## DevOps & 可观测性

CI/CD 流水线、Docker 与 K8s 容器化部署、日志聚合、指标监控与链路追踪，保障服务稳定运行。

## 测试

单元测试、集成测试、端到端测试——不同粒度的测试策略，确保代码质量与回归安全。
