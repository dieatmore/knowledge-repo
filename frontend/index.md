# 🖥️ 前端

前端开发知识体系，涵盖从浏览器底层原理到上层框架实践。

## 浏览器

从输入 URL 到页面呈现，理解渲染流水线的每个阶段：HTML 解析、CSS 计算、布局、绘制、合成。掌握重绘与回流的触发条件及优化策略，理解浏览器的缓存机制、跨域策略与安全模型。

## HTML & CSS

布局与样式是前端的基本功。从盒子模型出发，深入定位体系、Flex/Grid 布局、BFC 格式化上下文、响应式设计、CSS 动画等核心概念。既包括常见布局方案的横向对比，也涉及语义化、可访问性等工程素养。

## JavaScript

从原型链到事件循环，从闭包到异步编程——JS 的核心机制决定了代码怎么写才对。ES6+ 带来了模块化、箭头函数、解构等现代语法，TypeScript 进一步补全了类型系统，让大型项目更可控。

## Vue 生态

前端工程化的落脚点。围绕 Vue 框架，延伸出构建工具（Webpack / Vite）、包管理（npm / pnpm / yarn）、状态管理、路由等配套生态，形成完整的开发工具链。
